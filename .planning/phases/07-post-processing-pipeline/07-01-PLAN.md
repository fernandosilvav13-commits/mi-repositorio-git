# PLAN — Wire Post-Processing Pipeline into CVProcessor.process()

## Goal

Add a `_post_process()` method to `CVProcessor` that applies gender inference, phone normalization, RUT formatting, and name capitalization after LLM extraction. Wire it into `process()` so every extraction run through the pipeline gets consistent post-processing.

## Threat Model

```
Trust boundaries:
  ┌──────────────┐    process()     ┌───────────────┐
  │  LLM Output   │ ──────────────►  │  CVProcessor   │
  │  (Gemini)     │                  │  Post-Process  │
  └──────────────┘                  └───────────────┘
       Trust: semi-trusted           Trust: trusted

STRIDE Register:
  S/T Spoofing/Tampering:  LOW — infer_gender is a dict lookup on extracted
                           NOMBRES; normalize_phone and RUTFormatter use regex
                           on LLM-provided strings. No user-input injection
                           path beyond what LLM already consumes. → ACCEPT
  I   Info Disclosure:     LOW — gender inference is probabilistic; a false
                           positive (wrong gender) is a data quality issue,
                           not a security breach. → DOCUMENT as limitation
  R/D/E:                   N/A — no audit, no DoS vector, no privilege boundary
```

### Task 1 — Add _post_process() to CVProcessor and wire into process()

**Action:**

Add module-level helper in `cv_processor.py`:
```python
def _titlecase_name(name: str) -> str:
    """Title-case a name preserving diacritics."""
    if not name or name == "NO ENCONTRADO":
        return name
    return name.strip().title()
```

Add `_post_process` instance method to `CVProcessor`:
```python
    def _post_process(self, data: dict) -> dict:
        # Gender inference (only when LLM didn't find it — POST-04)
        genero = data.get("GENERO", "")
        if not genero or genero == "NO ENCONTRADO":
            nombres = data.get("NOMBRES", "")
            if nombres and nombres != "NO ENCONTRADO":
                data["GENERO"] = infer_gender(nombres)

        # Phone normalization (always — D-05 mandatory)
        if data.get("TELEFONO_CELULAR", "") and data["TELEFONO_CELULAR"] != "NO ENCONTRADO":
            _type, formatted = normalize_phone(data["TELEFONO_CELULAR"])
            if _type != "NO ENCONTRADO":
                data["TELEFONO_CELULAR"] = formatted
        if data.get("TELEFONO_FIJO", "") and data["TELEFONO_FIJO"] != "NO ENCONTRADO":
            _type, formatted = normalize_phone(data["TELEFONO_FIJO"])
            if _type != "NO ENCONTRADO":
                data["TELEFONO_FIJO"] = formatted

        # RUT formatting (always — success criteria #3)
        rut = data.get("RUT", "")
        if rut and rut != "NO ENCONTRADO":
            data["RUT"] = RUTFormatter.format(rut)

        # Name capitalization (always — D-04)
        for field in ("NOMBRES", "APELLIDOS"):
            val = data.get(field, "")
            if val and val != "NO ENCONTRADO":
                data[field] = _titlecase_name(val)

        return data
```

Wire into `process()` — add the line **after** experience enrichment and **before** `return data`:
```python
        data = self._post_process(data)
        return data
```

- Do NOT touch `get_top_3_experiences()` or the experience enrichment block
- Do NOT modify `gender_service.py`, `phone_service.py`, or `rut_formatter.py`

**Acceptance criteria:**
