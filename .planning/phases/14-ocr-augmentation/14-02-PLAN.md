### Task 2: LayoutResult schema

File: `backend/app/schemas/layout.py`

```python
class TextBlock(BaseModel):
    text: str
    bbox: tuple[int, int, int, int]  # x1, y1, x2, y2
    block_type: str  # "header", "body", "column"

class LayoutResult(BaseModel):
    blocks: list[TextBlock]
    full_text: str
    column_count: int
    reading_order: list[int]  # block indices in reading order
```

### Task 3: Column detection
- Collect all word x-coordinates
- Use clustering (k-means with k=1..3, pick best silhouette) to detect columns
- Assign each word to its column

### Task 4: Section/header detection
- Words with y-position near top of page → potential headers
- Words with larger font size (bbox height) → potential headers
- Group consecutive same-y words into lines

### Task 5: Reading order
- Sort blocks top-to-bottom, within same row left-to-right
- Multi-column: process left column top-to-bottom, then right column

## Verification
- `LayoutAnalyzer` returns structured blocks
- Column detection works for 1-column and 2-column layouts
- Reading order is natural
