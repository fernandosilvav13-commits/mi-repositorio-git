| Boundary | Description |

| YAML prompt files → PromptResolver | Prompt file content loaded as application configuration |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01 | Tampering | SectionBoundary start/end inversion | mitigate | Pydantic model_validator rejects end_line < start_line |
| T-10-02 | Tampering | NoiseRegion invalid type | mitigate | Pydantic model_validator restricts type to allowed enum set |
| T-10-03 | Spoofing | DetectionResult with extraneous LLM fields | mitigate | Pydantic ignores unknown fields by default (BaseModel default) |
| T-10-04 | Tampering | DetectionResult start_line negative | mitigate | Field-level ge=0 constraint on all integer line fields |
| T-10-05 | Denial of Service | Garbage LLM response with huge line numbers | accept | Validated by Pydantic — downstream must cap based on actual document length |
</threat_model>
